# 修改最后一次的提交信息

提交后发现描述信息有错误，进行修改（适用于修改最近一次提交信息） push前要仔细检查，push前要仔细检查，push前要仔细检查😂

  1. 输入以下命令
git commit --amend
  1. 进入vim编辑界面，修改提交信息，修改完后连按两次ZZ保存退出

  1. 再次推送（强制推送）
git push -f
