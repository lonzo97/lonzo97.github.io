    # 计算属性

    以字面量创建对象时，可以直接将 [] 的计算结果当做属性名

    var i = 0
var obj = {
    ['NO' + ++i]: i;
    ['NO' + ++i]: i;
    ['NO' + ++i]: i
}

// obj { NO1: 1, NO2: 2, NO3: 3 }
